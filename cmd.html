Команды для работы с файловой структурой
Команда Описание
mkdir [dirname] Создание директории. "-p dirname/{subdirname1,subdirname2}" - создает несколько папок с вложенностью
ls [dirname] Вывод поддиректорий и файлов текущей, или указанной директории
tree [dirname] Вывод файловой структуры в ввиде дерева
cd [dirname] Переход в указанную директорию. ".." - переход в родительскую директорию. "!$" - переход в только, что созданную директорию
touch [filename] Создание файла
"touch app/{css/main.css,js/main.js,index.html}" - создает несколько файлов
rm [filename] Удаление файла или директории. С ключем -r удаляет рекурсивно все поддиректории и дочернии файлы
clear Очистка консоли
mv Перемещение файлов
Команды для работы с git
Команда Описание
git init Инициалаизация git репозитория
git config чтение или установка настроек
git config --global core.autocrl true устанавливает правильное окончание строк в разных операционных системах
git config --global user.name "username" Глобальная установка имени разработчика
git config --global user.email "email@mail.ru" Глобальная установка почты разработчика
git help помощь по git
git status Показывает статус репозитория
git add . добавление неотслеживаемые файлы в файлы готовые к коммиту
git commit -m "описание коммита" делает коммит (т.е. состояние сохранено и зафиксировано)
git log просмотр истории статусов
git chckout HEAD index.html отменяет не принятые к коммиту файлы
git reset HEAD index.html отменяет добавленные к коммиту файлы и переносит их как непринятые к коммиту
git checkout -- . возвращает репозиторий к состоянию последнего коммита
